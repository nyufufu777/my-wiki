# 技術別ノート

このカテゴリには、言語、フレームワーク、開発ツールを使うための解説を置きます。

## 置くページ

- Docker、Dockerfile、Docker Compose、Nginx
- Python、FastAPI、SQLAlchemy、Alembic
- React、TypeScript、Ant Design、i18n
- PostgreSQL、Playwright、GitHub Actions

## 置かないページ

特定プロジェクトだけのディレクトリ構成や設定値は [プロジェクト解説](../projects/README.md) に置きます。

## 記事の目標

初めて読む人が「何のための技術か」「最小限どう動かすか」「設定やコードをどうレビューするか」を順に理解できることを目標にします。
