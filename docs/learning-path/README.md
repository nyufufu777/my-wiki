# 学習の進め方

このカテゴリには、「何をどの順番で学ぶか」と「今どこまで理解できているか」を置きます。

## 置くページ

- 技術スタックを学ぶ順番
- 学習ロードマップ
- 学習ログと振り返り
- 次に調べるテーマ

## 置かないページ

DockerやHTTPなど一つの技術を詳しく説明する記事は、[技術別ノート](../technologies/README.md) または [基礎概念](../concepts/README.md) に置きます。

## 読み方

新しい技術を始める前に、このカテゴリで目的と到達点を確認します。学習後は、分かったことだけでなく、まだ説明できないことも記録します。次に読むページやIssueを決めやすくなります。
